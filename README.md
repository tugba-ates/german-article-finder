# German Article Finder 🇩🇪

A web application that helps you find the correct German article 
(der/die/das) for any German noun.

## 🔗 Live Demo
[Click here to try it](https://tugba-ates.github.io/german-article-finder)

## Features
- 115,339 word database
- Instant search results
- Color-coded results (blue = der, pink = die, green = das)
- Live English translation

## Tech Stack
- HTML, CSS, JavaScript
- No frameworks, no dependencies
- Data parsed from Wiktionary
